    更換主題

    下載到theme資料夾

    $ git clone https://github.com/yelog/hexo-theme-3-hexo.git themes/3-hexo

    修改 blog目錄 _config.yml 中的:

    theme: 3-hexo

    搭配主題的文字計數器

    blog目錄下
    npm i --save hexo-wordcount

    404頁面

    Blog資料夾下
    hexo new page 404
